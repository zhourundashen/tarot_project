### 问题4: 错误处理不友好

**症状**:
- 点击发送后显示"抱歉，我暂时无法回应"
- 没有详细的错误信息

**原因**: 错误被catch但没有显示详细信息

**修复方案**: 增强错误处理

**文件**: `frontend/js/app.js`

**找到这段代码** (约885-911行):
```javascript
async sendChatMessage() {
    const input = document.getElementById('chat-input');
    const message = input ? input.value.trim() : '';
    
    if (!message) return;
    
    input.value = '';
    this.addUserMessage(message);
    
    this.showTypingIndicator();
    
    try {
        const response = await TarotAI.chatWithContext(
            message,
            this.userQuestion,
            this.currentSpread,
            this.drawnCards,
            this.chatMessages
        );
        
        this.hideTypingIndicator();
        this.addAIMessage(response);
    } catch (error) {
        this.hideTypingIndicator();
        this.addAIMessage('抱歉，我暂时无法回应。请稍后再试。');
    }
}
```

**修改为**:
```javascript
async sendChatMessage() {
    const input = document.getElementById('chat-input');
    const message = input ? input.value.trim() : '';
    
    if (!message) {
        this.showToast('请输入消息', 'warning');
        return;
    }
    
    input.value = '';
    this.addUserMessage(message);
    
    this.showTypingIndicator();
    
    try {
        // 检查AI是否可用
        if (typeof TarotAI === 'undefined') {
            throw new Error('AI模块未加载，请刷新页面');
        }
        
        // 检查必要的参数
        if (!this.currentSpread) {
            console.warn('currentSpread为空，使用默认值');
        }
        
        const response = await TarotAI.chatWithContext(
            message,
            this.userQuestion || '',
            this.currentSpread || { name: '未知牌阵' },
            this.drawnCards || [],
            this.chatMessages || []
        );
        
        this.hideTypingIndicator();
        this.addAIMessage(response);
        
    } catch (error) {
        this.hideTypingIndicator();
        
        // 详细的错误信息
        console.error('聊天错误:', error);
        
        let errorMessage = '抱歉，我暂时无法回应。';
        
        if (error.message.includes('NetworkError') || error.message.includes('Failed to fetch')) {
            errorMessage = '网络连接失败，请检查后端服务是否启动。';
        } else if (error.message.includes('AI模块未加载')) {
            errorMessage = 'AI模块未加载，请刷新页面重试。';
        } else if (error.message.includes('API')) {
            errorMessage = 'AI服务暂时不可用，请稍后再试。';
        } else {
            errorMessage = `错误: ${error.message}`;
        }
        
        this.addAIMessage(errorMessage);
        this.showToast(errorMessage, 'error');
    }
}
```

### 问题5: 后端API密钥未配置

## ⚠️ 常见陷阱

### 陷阱1: 浏览器缓存
**症状**: 修改代码后没有生效  
**解决**: Ctrl+Shift+Delete 清除缓存，或 Ctrl+F5 硬刷新

### 陷阱2: 端口被占用
**症状**: 后端启动失败  
**解决**: 
```bash
# 查找占用进程
netstat -ano | findstr :3000

# 杀掉进程（替换PID）
taskkill /PID 12345 /F
```

### 陷阱3: 跨域问题
**症状**: CORS错误  
**解决**: 后端已配置CORS，确保使用 `http://localhost:3000` 而不是 `file://`

### 陷阱4: 空值问题
**症状**: "Cannot read property 'name' of undefined"  
**解决**: 代码中已添加空值检查，确保参数都有默认值
