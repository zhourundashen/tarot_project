## 项目概览

### 项目名称
**神秘塔罗 - 智能塔罗占卜平台**

### 项目简介
一款基于AI的在线塔罗占卜应用，提供22种专业牌阵、78张塔罗牌解读服务，支持深度AI解读、追问互动、历史记录等功能。

### 核心特性
- ✨ 22种专业牌阵（从单牌阵到凯尔特十字）
- 🎴 78张韦特塔罗牌（大阿卡纳22张 + 小阿卡纳56张）
- 🤖 AI智能解读（智谱GLM-5）
- 📊 智能知识库系统（按需加载）
- 🎨 精美分享功能（4种风格模板）
- 📱 移动端完美适配
- 👤 完整用户系统（注册/登录/JWT认证）
- 💰 积分系统（签到/兑换/流水）
- 🎵 音效系统（洗牌/抽牌/翻牌）
- 📖 新手引导（5步教程）

### 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | HTML5 + CSS3 + JavaScript (原生ES6+) |
| **后端** | Node.js + Express |
| **数据库** | MySQL 5.7+ |
| **AI服务** | 智谱GLM-5 API |
| **认证** | JWT (JSON Web Token) |
| **架构** | MVC模式 |
| **文档** | Swagger API文档 |

### MVC架构说明

**后端采用标准MVC架构**：

1. **Model (数据模型层)**
   - 负责数据库操作
   - 封装SQL查询
   - 提供数据接口

2. **View (视图层)**
   - 前端HTML页面
   - CSS样式
   - JavaScript交互

3. **Controller (控制器层)**
   - 处理业务逻辑
   - 协调Model和Response
   - 验证和错误处理

4. **Routes (路由层)**
   - 定义API端点
   - 映射到Controller
   - 应用中间件

5. **Middleware (中间件层)**
   - 认证检查
   - 数据验证
   - 错误处理
   - 安全防护

   ↓

   ↓
