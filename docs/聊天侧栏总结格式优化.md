# 聊天侧栏总结格式优化 - 移除长分隔线

## 问题描述
AI生成的总结中包含了长分隔线 `══════`，导致聊天侧栏显示很长，占用空间。

## 解决方案
在 `app.addAIMessage()` 方法中，如果检测到长分隔线（═══），就自动移除。

## 修改位置
文件: `frontend/js/app.js`
方法: `addAIMessage()`

## 修改内容
在 `addAIMessage()` 方法中添加分隔线过滤逻辑。

## 测试
- 刷新页面
- 生成总结
- 查看聊天侧栏
- 确认没有长分隔线

## 注意
- 复制和下载功能仍然保留完整格式（有分隔线）
- 只有聊天侧栏显示时移除分隔线
