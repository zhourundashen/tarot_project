================================================================================
                              API配置说明
================================================================================

【当前配置】千问Plus API
- 提供商：阿里云 DashScope
- 模型：qwen-plus
- API Key：已配置在 js/config.js

【切换API】
编辑 js/config.js，修改以下字段：
- apiKey: 新的API密钥
- apiUrl: 新的API地址
- model: 新的模型名称

支持的API：
- 阿里云千问（当前）
- DeepSeek
- OpenAI
- 智谱AI (GLM)

================================================================================
                              使用说明
================================================================================

1. 用浏览器打开 index.html
2. 点击「开始占卜」
3. 输入问题（或跳过）
4. 选择牌阵
5. 洗牌、切牌
6. 抽牌
7. 等待AI解读
8. 满意→保存，有问题→继续追问
9. 最终生成总结
