===============================================================================
                          塔罗牌占卜网页项目 - 四周任务总览
===============================================================================

项目目标：打造一个精美的塔罗牌占卜网页，包含AI解读、多主题、牌阵功能

技术栈：Vue 3 + TypeScript + Vite + Pinia + Tailwind CSS
操作系统：Windows
每日投入：3-4小时


===============================================================================
                                第一周任务
===============================================================================

【学习阶段】前端基础 + 项目环境搭建

Day 1-2: 前端基础速成（HTML/CSS/JavaScript）
---------------------------------------------
学习重点：
- HTML基础标签：div, span, p, button, img, input
- CSS：Flexbox布局、颜色、定位、盒模型
- JavaScript：
  * 变量声明（let/const）
  * 箭头函数 () => {}
  * 解构赋值 {a, b} = obj
  * 模板字符串 `${a}`
  * 数组方法：map, filter, forEach
  * async/await 异步操作

练习任务：
1. 创建个人介绍HTML页面
2. 用CSS美化布局（Flexbox）
3. 实现按钮点击变色交互

验收标准：
□ 能独立创建.html/.css/.js文件
□ 理解F12开发者工具基本使用
□ 能用fetch发送简单HTTP请求

学习资源：
- MDN Web 文档
- 菜鸟教程


Day 3-4: Vue 3 核心概念
---------------------------------------------
学习重点：
- 响应式系统（ref, reactive）
- 模板语法 {{ 变量 }}
- 指令：v-if, v-for, @click, :class
- 计算属性 computed
- 生命周期：created, mounted
- 组件化（props, emit）

练习任务：
1. 创建计数器组件
2. 实现待办事项列表（增删查）

验收标准：
□ 理解ref/reactive的区别
□ 能创建.vue单文件组件
□ 掌握父子组件通信

学习资源：
- Vue 3 中文文档：https://cn.vuejs.org/
- B站：尚硅谷Vue3教程


Day 5-6: TypeScript 基础
---------------------------------------------
学习重点：
- 基本类型：string, number, boolean
- 接口定义 interface
- 可选属性 ?:
- 联合类型 |
- Vue中的类型使用

示例：
interface Card {
  name: string
  position: 'upright' | 'reversed'
}

练习任务：
1. 为待办事项应用添加TypeScript类型
2. 定义用户登录接口

验收标准：
□ 能定义和使用interface
□ 理解联合类型、可选类型
□ Vue组件中正确使用类型注解

学习资源：
- TypeScript 中文文档：https://www.tslang.cn/


Day 7: 项目环境搭建
---------------------------------------------
任务清单：
1. 创建Vue 3 + TypeScript项目
   npm create vite@latest tarot-reading -- --template vue-ts

2. 安装依赖
   cd tarot-reading
   npm install

3. 启动项目
   npm run dev

4. 项目结构初始化
   创建文件夹：
   - src/components/（组件）
   - src/types/（类型定义）
   - src/data/（数据层）

5. 准备塔罗牌素材
   - 22张大阿卡纳牌图片（牌面+牌背）
   - 图片命名：0-愚人.jpg, 1-魔术师.jpg, ..., 21-世界.jpg

6. 创建基础文件
   - src/types/card.ts（塔罗牌类型定义）
   - src/data/cards.ts（塔罗牌数据）
   - src/components/TarotCard.vue（单张卡牌组件）

验收标准：
□ 项目能正常启动（npm run dev）
□ 热更新工作正常
□ 能渲染一个简单的塔罗牌组件

Day 14: 塔罗牌数据完善
---------------------------------------------
任务清单：
1. 完成78张塔罗牌数据
   - 22张大阿卡纳（0-愚人到21-世界）
   - 56张小阿卡纳（权杖、圣杯、宝剑、星币，各14张）

2. 每张牌包含字段：
   - id: 唯一标识
   - number: 编号
   - type: 'major' | 'minor'
   - suit: 'Major' | 'Wands' | 'Cups' | 'Swords' | 'Pentacles'
   - name: 中文名称
   - nameEn: 英文名称
   - keywords: 关键词数组
   - upright: 正位含义和关键词
   - reversed: 逆位含义和关键词
   - symbolism: 象征意义

3. 图片素材整理
   - 确保所有牌面图片齐全
   - 检查图片质量

验收标准：
□ 78张牌数据完整
□ 所有图片文件存在
□ 数据类型正确
