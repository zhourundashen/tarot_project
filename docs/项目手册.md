# 塔罗占卜Web应用 - 项目手册

## 一、项目概述

### 1.1 项目名称
神秘塔罗 - 智能塔罗占卜平台

### 1.2 项目简介
一款基于AI的在线塔罗占卜应用，提供22种专业牌阵、78张塔罗牌解读服务，支持深度AI解读、追问互动、历史记录等功能。

### 1.3 技术栈
- 前端：HTML5 + CSS3 + JavaScript (原生)
- AI服务：智谱AI GLM-4
- 存储：localStorage（后续升级为数据库）
- 部署：静态托管 / 云服务器

### 1.4 目标用户
- 塔罗爱好者
- 对占卜感兴趣的用户
- 寻求生活指引的人群

    userId: "string",

    userId: "string",

    balanceAfter: 30,      // 操作后余额
    createdAt: "datetime"

```

```
POST /api/reading/create    # 创建占卜记录

POST /api/reading/ai        # AI解读
```

```
