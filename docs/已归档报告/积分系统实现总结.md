## 💾 数据库设计

### users表（已修改）
```sql
ALTER TABLE users 
ADD COLUMN total_credits INT DEFAULT 50 COMMENT '累计获得积分';
```

### sign_records表（新增）
```sql
CREATE TABLE sign_records (
    id BIGINT AUTO_INCREMENT PRIMARY KEY,
    user_id VARCHAR(64) NOT NULL,
    sign_date DATE NOT NULL,
    credits_earned INT DEFAULT 10,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    UNIQUE KEY uk_user_date (user_id, sign_date)
);
```

### credit_transactions表（新增）
```sql
CREATE TABLE credit_transactions (
    id VARCHAR(64) PRIMARY KEY,
    user_id VARCHAR(64) NOT NULL,
    type VARCHAR(20) NOT NULL,
    amount INT NOT NULL,
    balance_after INT NOT NULL,
    description VARCHAR(255),
    related_id VARCHAR(64),
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 🔌 API接口

### 1. 签到
```
POST /api/credits/sign
Headers: Authorization: Bearer {token}

Response:
{
    "success": true,
    "message": "签到成功",
    "data": {
        "creditsEarned": 10,
        "currentCredits": 60,
        "totalCredits": 60
    }
}
```

### 2. 获取签到状态
```
GET /api/credits/status
Headers: Authorization: Bearer {token}

Response:
{
    "success": true,
    "data": {
        "hasSignedToday": false,
        "currentCredits": 50,
        "totalCredits": 50,
        "creditsPerSign": 10,
        "creditsPerReading": 20
    }
}
```

### 3. 积分兑换
```
POST /api/credits/exchange
Headers: Authorization: Bearer {token}

Response:
{
    "success": true,
    "message": "兑换成功",
    "data": {
        "creditsUsed": 20,
        "currentCredits": 30
    }
}
```

### 4. 积分历史
```
GET /api/credits/history?page=1&limit=20
Headers: Authorization: Bearer {token}

Response:
{
    "success": true,
    "data": {
        "transactions": [...],
        "pagination": {
            "page": 1,
            "limit": 20,
            "total": 5,
            "totalPages": 1
        }
    }
}
```

### 5. 检查积分
```
GET /api/credits/check
Headers: Authorization: Bearer {token}

Response:
{
    "success": true,
    "data": {
        "canRead": true,
        "needCredits": false,
        "currentCredits": 50,
        "requiredCredits": 20
    }
}
```

## 🚀 使用流程

## 📊 数据统计

### 数据表
- `users`: 用户表（积分字段）
- `sign_records`: 签到记录表
- `credit_transactions`: 积分流水表
- `reading_records`: 占卜记录表（消耗积分字段）

### 查询示例
```sql
-- 查询用户积分
SELECT credits, total_credits FROM users WHERE id = ?;

-- 查询今日签到
SELECT * FROM sign_records 
WHERE user_id = ? AND sign_date = CURDATE();

-- 查询积分流水
SELECT * FROM credit_transactions 
WHERE user_id = ? 
ORDER BY created_at DESC 
LIMIT 20;
```
