## 📖 技术文档

### 断点系统
```css
/* 移动端 */
@media (max-width: 768px) { }

/* 超小屏幕 */
@media (max-width: 360px) { }

/* 横屏 */
@media (max-width: 768px) and (orientation: landscape) { }
```

### 触摸区域规范
- 最小尺寸：44x44px
- 最小padding：12px
- 最小font-size：14px

### 性能优化技巧
1. 使用`transform`代替`top/left`
2. 使用`will-change`启用GPU加速
3. 减少DOM操作
4. 避免强制同步布局
5. 使用CSS动画代替JS动画
