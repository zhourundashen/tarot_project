# 三牌阵中间牌居中显示修复报告

**修复日期**: 2026-03-02  
**问题**: 所有横向排列的三牌阵的中间那张牌没有居中展示

---

## 🔍 问题分析

### 受影响的牌阵
1. **三牌阵** (`three-card`) - 过去、现在、未来
2. **抉择牌阵** (`three-card-choice`) - 选项A、选项B、建议
3. **身心灵牌阵** (`mind-body-spirit`) - 身体、心理、灵性

### 问题描述
原布局使用简单的等间距排列，没有让中间的牌真正居中：

**原代码**:
```javascript
getPositionStyle: (i) => `left: ${(gap + i * (cardW + gap)) * scale}px; top: 50%; transform: translate(-50%, -50%);`
```

**原布局位置**:
- 第0张牌：left = 20px
- 第1张牌：left = 170px
- 第2张牌：left = 320px

**问题**: 中间牌（第1张）不在容器中心，左右不对称

---

## ✅ 修复方案

### 新的布局逻辑

**核心思路**:
1. 使用CSS的`calc()`函数和百分比定位
2. 中间牌使用`left: 50%`确保在容器中心
3. 左右牌使用`calc(50% ± offset)`对称分布

**最终代码**:
```javascript
getPositionStyle: (i) => {
    const positions = [
        `left: calc(50% - ${(cardW + gap) * scale}px); top: 50%; transform: translate(-50%, -50%);`,
        `left: 50%; top: 50%; transform: translate(-50%, -50%);`,
        `left: calc(50% + ${(cardW + gap) * scale}px); top: 50%; transform: translate(-50%, -50%);`
    ];
    return positions[i];
}
```

**新布局位置** (相对于容器宽度):
- 第0张牌：left = calc(50% - 150px)
- 第1张牌：left = 50% (完美居中)
- 第2张牌：left = calc(50% + 150px)

**效果**: 中间牌在容器中完美居中，左右对称，无论容器实际宽度如何

### 为什么使用百分比定位？

**问题分析**:
- `.spread-display` 容器有 `min-width: 600px`
- JavaScript设置的布局宽度为470px（3张牌+间距）
- 如果使用固定像素值，牌会相对于470px居中
- 但容器实际宽度可能是600px或更大，导致牌不居中

**解决方案**:
- 使用`left: 50%`让中间牌始终在容器的中心
- 使用`calc(50% ± offset)`让左右牌对称分布
- 无论容器实际宽度是多少，牌都会正确居中

### 修改内容

#### 1. 三牌阵 (`three-card`) - 第1987-1998行

**修改前**:
```javascript
'three-card': {
    width: `${(cardW * 3 + gap * 4) * scale}px`,
    height: `${cardH * scale + 40}px`,
    getPositionStyle: (i) => `left: ${(gap + i * (cardW + gap)) * scale}px; top: 50%; transform: translate(-50%, -50%);`

```

**修改后**:
```javascript
'three-card': {
    width: `${(cardW * 3 + gap * 4) * scale}px`,
    height: `${cardH * scale + 40}px`,
    getPositionStyle: (i) => {
        const centerX = (cardW * 3 + gap * 4) * scale / 2;
        const offset = (cardW + gap) * scale;
        const positions = [
            `left: ${centerX - offset}px; top: 50%; transform: translate(-50%, -50%);`,
            `left: ${centerX}px; top: 50%; transform: translate(-50%, -50%);`,
            `left: ${centerX + offset}px; top: 50%; transform: translate(-50%, -50%);`
        ];
        return positions[i];
    }
},
```

## 📐 布局计算公式

### 容器宽度计算
```
容器宽度 = (cardW * 3 + gap * 4) * scale
         = (130 * 3 + 20 * 4) * 1
         = (390 + 80) * 1
         = 470px
```

### 中心点计算
```
centerX = 容器宽度 / 2
        = 470 / 2
        = 235px
```

### 牌间距
```
offset = (cardW + gap) * scale
       = (130 + 20) * 1
       = 150px
```

### 最终位置
```
左牌: left = 235 - 150 = 85px
中牌: left = 235px (居中)
右牌: left = 235 + 150 = 385px
```
