## 🛠️ 技术实现

### 1. 创建懒加载工具模块

**文件**: `js/lazy-loader.js` (新建)

**核心功能**:
```javascript
const LazyLoader = {
    observer: null,              // Intersection Observer实例
    
    init() {},                   // 初始化Observer
    observe(imgElement) {},      // 观察图片元素
    loadImage(imgElement) {},    // 加载单张图片
    createPlaceholderSVG() {},   // 创建SVG占位符
    setupLazyImage(img, src) {}, // 设置懒加载图片
    loadAll() {},                // 加载所有图片
    disconnect() {}              // 断开观察
}
```

**关键特性**:
- 使用 `Intersection Observer API` 监听图片进入视口
- 提前50px预加载（`rootMargin: '50px 0px'`）
- SVG占位符（🔮图标）
- 平滑的淡入动画效果
- 兼容不支持Observer的浏览器（直接加载）

## 🎯 工作原理

### 懒加载流程

```
1. 页面加载
   ↓
2. 创建图片元素，设置 data-src（不加载）
   ↓
3. LazyLoader.setupLazyImage() 设置占位符
   ↓
4. Intersection Observer 监听图片进入视口
   ↓
5. 图片进入视口前50px
   ↓
6. 触发 loadImage()
   ↓
7. 加载真实图片
   ↓
8. 替换占位符，添加淡入动画
```

### Intersection Observer 配置

```javascript
{
    rootMargin: '50px 0px',  // 提前50px加载
    threshold: 0.01          // 1%可见即触发
}
```

**优势**:
- 不需要滚动事件监听
- 性能更好
- 自动处理元素可见性
